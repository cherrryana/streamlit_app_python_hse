# Temperature Analysis & Monitoring

Streamlit приложение для анализа температур и мониторинга текущей погоды через OpenWeatherMap API

## Ссылка на Streamlit приложение

https://pythonhw1yavolkova.streamlit.app/

## Как запустить?

```bash
streamlit run app.py
```

## Файлы и папки

- `app.py` - Streamlit приложение
- `analysis.py` - анализ временных рядов (выводы по параллелизму в комментариях)
- `weather_monitor.py` - мониторинг погоды (выводы по async/sync в комментариях)
- `temperature_data.csv` - исторические данные
- `/graphics_examples` - примеры графиков на нескольких городах
